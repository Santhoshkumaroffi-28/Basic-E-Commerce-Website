# ShopEasy – E-Commerce Web Application

ShopEasy is a simple and responsive e-commerce web application built using basic HTML, CSS, and JavaScript.
This project demonstrates product listing, shopping cart functionality, form validations, and a basic checkout system.

# Features Included:

### 1. Navigation Menu:

* Home
* Products
* Contact Us
* About Us

Provides smooth navigation across the website sections.

### 2. Product Display Section:

* 3 sample products with images
* Price display
* "Add to Cart" button
* Responsive product layout (flexbox)

### 3. Shopping Cart:

* Items added dynamically using JavaScript
* Auto-updated total price
* Cart displayed in a separate box
* "Proceed to Checkout" button
  
### 4. Checkout Page:

* Appears only if the cart is not empty
* Takes user details:

  * Full Name
  * Address
  * Phone Number
* Validates all required fields
* Shows success message after placing order

### 5. User Login Validation:

* Email and password fields
* Checks:
  * Empty fields
  * Valid email format
* Displays success or error message

### 6. User Registration Validation:

* Fields:
  * Full Name
  * Email
  * Password
  * Confirm Password

* Validations:
  * All fields must be filled
  * Password length must be ≥ 6
  * Passwords must match

* Displays success or error message
   
### 7. Responsive Design:

* Uses media queries
* Adjusts product cards for mobile screens
* Simple user-friendly layout

## Technologies Used:

* HTML5 — for structure
* CSS3 — for styling and responsiveness
* JavaScript — for:

  * Cart management
  * Checkout operation
  * Login & register form validations

## How to Run the Project:

1. Download or copy all the project files.
2. Save the HTML code as 'index.html'.
3. Open the file in any browser (Chrome, Firefox, Edge, etc.).
4. The website will load instantly — no extra setup required.

## Project Highlights:

* Perfect for beginners learning HTML, CSS & JavaScript
* Demonstrates real-world e-commerce concepts
* Fully functional frontend project suitable for college assignments
